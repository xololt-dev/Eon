## Build
Developed on Ubuntu 22.04

Requires SDL2, GLM, googletest

Uses CMake and ninja

#### Debug
```
--build /home/xololt/repos/Eon/build --config Debug --target all --
```
#### Release
```
--build /home/xololt/repos/Eon/build --config Release --target all --
```
